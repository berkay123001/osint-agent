Bölüm 05

Yeni Tool'lar & Entegrasyonlar

🗺️ GPX Analyzer Yeni

GPS/GPX dosyası analizi. Hotspot tespiti, haversine ile coğrafi merkez, Nominatim reverse geocoding. Eyfel Kulesi OSINT Challenge başarıyla çözüldü.

📔 Obsidian Sync Yeni

Tüm ajan çıktıları otomatik Obsidian vault'una yazılır. obsidian_write, obsidian_append, obsidian_search, obsidian_daily_log.

🎓 Plagiarism Checker Yeni

Akademik intihal/özgünlük analizi. arXiv + Semantic Scholar çift kaynak, yazar profili çıkarma.

🕸️ Graph UI Geliştirildi

Neo4j veritabanı için interaktif browser (npm run graph, port 3333). Force-directed layout, renkli node tipleri.

🏷️ ML Label Sistemi Yeni

GNN eğitimi için soft labeling. mark_false_positive tool'u ile negatif/pozitif örnekler Cypher sorgusu ile çekilebilir.

⚠️ Claim / Fact Grafı Yeni

Fact-check döngüsü için Claim, Fact, Source node tipleri eklendi. IOC ve tehdit aktörü grafı da hazır.
